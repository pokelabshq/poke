# Poke — Your code, maintained autonomously.

Poke is an autonomous AI DevOps agent that monitors your repos, fixes bugs, updates dependencies, reviews PRs, and ships code while you sleep.

## Install

```bash
npm install -g poke
npx poke init
```

## Works with

- GitHub
- TypeScript, Python, Go, Rust, Ruby, Java, and more
- npm, pip, cargo, go mod, bundler — any package manager
