🧑‍💻 About

Poke Labs is a one-person open source studio run by Alexander Wondwossen, a 13-year-old developer in Toronto, Canada.

We believe the future of software is autonomous. AI agents should handle the boring stuff — dependency updates, repo maintenance, monitoring — so humans can focus on building.

Everything we build is MIT licensed and free to use. No VC money, no ads, no tracking. Just good software.
