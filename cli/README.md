# ai-council

> Multi-agent deliberation in your terminal

Ask a question. Get perspectives from 5 AI personas.

## Install

```bash
npm install -g ai-council
```

## Use

```bash
ai-council "Should I use Postgres or SQLite for my side project?"
ai-council "Is now the right time to launch?"
ai-council --status
```

## Personas

- 🎯 **The Strategist** — market fit, positioning, moats
- 🚀 **The Hustler** — growth, shipping, scrappy tactics
- 🔧 **The Engineer** — feasibility, architecture, best practices
- 💀 **The Critic** — devil's advocate, finds flaws
- 🌿 **The Visionary** — big picture, future trends
